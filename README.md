# onlineStoreProject
Simple online store with backend and working Admin panel for add new items.
In this project used:

 Backend:
  -----------
  - Node JS
  - Express
  - PostgreSQL
  - jwt authentication
  
 Frontend:
  -----------
  - React JS
  - Mobx
  - axios
  - react route v6
  - jwt authentication
  - handmade reusebale Ui-components
